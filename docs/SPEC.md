# CyberAI Prototype Specification

CyberAI is a responsive employee cybersecurity learning platform. The prototype
shows an eight-module roadmap and implements one complete vertical slice:
Phishing: The Lure (Email).

## Core contract

- Email/password authentication and isolated anonymous demo sessions.
- Four-question phishing quiz with server-controlled scoring.
- Confidence is required before submission and is used for insights only.
- Best attempt drives progress; 350/500 points passes the module.
- Passing unlocks the phishing badge and cheat sheet.
- Seven future modules and the certificate use polished locked states.
- SQL migrations are the canonical content and schema source.
